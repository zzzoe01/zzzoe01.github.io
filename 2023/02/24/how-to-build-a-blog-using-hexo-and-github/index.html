how to build a blog using hexo and github

一、工具下载

  1. Node.js
    下载链接： Download | Node.js (nodejs.org)
    安装教程： Node.js安装及环境配置之Windows篇 - 刘奇云 - 博客园 (cnblogs.com)

  2. 下载git
    下载链接：Git - Downloading Package (git-scm.com)
    安装教程：Windows系统Git安装教程（详解Git安装过程） - 学为所用 - 博客园 (cnblogs.com)
